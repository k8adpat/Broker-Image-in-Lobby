## Dependencies & Instructions
**Dependencies:**
[SuperBLT](https://superblt.znix.xyz)

**Instructions:**
Make sure to first have SuperBLT installed.

Optional video tutorial for installing SuperBLT, BeardLib, and mod_overrides mods: https://youtu.be/rin_sjSJIkg

## Instalation
Go to yours payday 2 mods folder. If you dont know where is mods folder - use [this guide](https://wiki.modworkshop.net/books/installing-payday-2-mods/page/finding-payday-2s-install-folder)

Extract your zip into `mods`

*or*
```cmd
# use this inside of PAYDAY 2 folder
# /path/to/your/steam/..../Steam/steamapps/common/PAYDAY 2/
mkdir mods
cd mods
git clone https://github.com/k8adpat/Broker-Image-in-Lobby.git
```
